 POWER-BI---Laptop-Market-Insights
Interactive Power BI dashboard analyzing laptop market trends and performance metrics. Includes data cleaning, transformations, and DAX-based KPIs to derive meaningful business insights.

This project presents an interactive Power BI dashboard that analyzes laptop market trends, pricing patterns, and key performance indicators (KPIs). 
The project includes essential data transformations, DAX measures, and visuals to derive actionable business insights.  

 Overview  
- Cleaned and transformed the dataset using Power Query Editor.  
- Removed unnecessary columns and standardized data for analysis.  
- Created calculated columns, measures, and KPIs using DAX.  
- Designed an interactive dashboard highlighting laptop market insights through KPIs and visual comparisons.  
- Derived insights on pricing trends, performance factors, and customer preferences.  

Tools Used  
- Power BI Desktop  

 Files Included 
- PowerBI_Laptop_Market_Insights_Report.pdf — Complete report describing the analysis, transformations, and insights.  
- PowerBI_Laptop_Market_Insights_Presentation.pptx — Presentation summarizing dashboard visuals and findings.
